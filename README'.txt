1. install python == buka cmd ketik pip install python atau install langsung dari browser
2. Selanjutnya install pyqt5 == buka cmd ketik: 
pip install pyqt5
pip install pyqt5-tools
pip install pymysql
3. buka xampp dan start apache & mysql, dan buka browser ketik localhost/phpmyadmin/
4. import database dbmaba (jika tidak bisa, coba buat database baru dengan nama dbmaba, kemudian import dbmaba.sql)
5. selesai
